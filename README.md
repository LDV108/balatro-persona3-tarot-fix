# Persona 3 Tarots [FIXED VERSION]

> Original mod by [Salt_of_Eden](https://www.nexusmods.com/balatro/mods/70). Fixed for Steamodded 1.x by LDV.

Replaces all 22 Major Arcana tarot card sprites with artwork from Persona 3 Reload.

Pairs well with [TarotLabels](https://www.nexusmods.com/balatro/mods/870), my own mod that overlays card names on the art since the Persona 3 skin doesn't print them.

## Requirements

- Balatro
- Steamodded (SMODS 1.x)

## Install

Drop the `Persona 3 Tarot card-70-1-0-1732879781` folder into:

```
%AppData%\Roaming\Balatro\Mods\
```

## What was fixed

- Rewrote the Lua for the modern SMODS 1.x API
- Added `metadata.json` (required by SMODS 1.x)
- Fixed asset folder structure
